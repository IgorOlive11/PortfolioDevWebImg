Descrição Tecnológica.

Esta página é destinada a descrever algumas das tecnologias utilizadas por mim. Aqui você encontrará uma visão abrangente das linguagens de programação frameworks e APIs ultilizadas em meus projetos.

HTML.

Prós do HTML5:

  • O HTML5 introduziu uma série de novas tags semânticas. Essas tags permitem uma melhor estruturação e organização do conteúdo, facilitando a compreensão pelos motores de busca e melhorando a acessibilidade.
  • Trás recursos nativos para reprodução de vídeo e áudio sem a necessidade de plugins externos, como o Adobe Flash. Isso simplifica a experiência do usuário e torna mais fácil a incorporação de mídia em sites e aplicativos web.
  • É suportado por todos os principais navegadores modernos. Isso significa que os desenvolvedores podem criar sites e aplicativos que funcionam em várias plataformas e dispositivos diferentes, incluindo computadores, smartphones e tablets.

Limitações do HTML5:

  • Embora os navegadores modernos ofereçam bom suporte ao HTML5, ainda existem diferenças sutis na implementação de certos recursos. Isso pode levar a problemas de compatibilidade e requerer soluções alternativas para garantir a consistência em diferentes navegadores.
  • O HTML5 trouxe uma gama de recursos avançados, mas isso também significa que o aprendizado e a implementação desses recursos podem ser mais complexos. Desenvolvedores podem enfrentar curvas de aprendizado íngremes para utilizar todas as funcionalidades disponíveis.
  • Com o aumento das interações dinâmicas e recursos avançados, o HTML5 pode introduzir possíveis vulnerabilidades de segurança se não for utilizado corretamente. É importante estar ciente das boas práticas de segurança ao desenvolver com HTML5.

O que é HTML5?
HTML5 é uma linguagem de marcação de tecnologia central da Internet usada para estruturar e apresentar conteúdo para a World Wide Web.
HTML
HTML5 é a quinta e mais recente versão da linguagem de marcação HTML (Hypertext Markup Language). Ela foi lançada em 2014 e trouxe consigo diversas melhorias e recursos avançados para o desenvolvimento de sites e aplicativos web.

Grandes projetos que utilizam HTML5:
Google Docs, YouTube, Spotify, Netflix, Adobe Photoshop Express, Slack.

CSS.

Prós do CSS3:

  • O CSS3 oferece uma ampla gama de recursos de estilização e layout, permitindo aos desenvolvedores maior controle sobre a aparência de um site ou aplicativo. É possível criar designs personalizados e responsivos, adaptando-se a diferentes tamanhos de tela e dispositivos.
  • Com recursos como transições, animações, gradientes e sombras, o CSS3 possibilita a criação de efeitos visuais atrativos e interativos, tornando a experiência do usuário mais envolvente e dinâmica.
  • O CSS3 permite a criação de estilos mais eficientes e otimizados, resultando em páginas web que carregam mais rapidamente. Além disso, ele suporta a criação de layouts flexíveis e responsivos, melhorando a experiência em diferentes dispositivos.

Limitações do CSS3:

  • Nem todos os recursos do CSS3 são suportados por navegadores mais antigos. Isso pode exigir a implementação de soluções alternativas ou a limitação do uso de determinados recursos para garantir uma experiência consistente em diferentes navegadores.
  • O CSS3 trouxe uma série de recursos avançados, o que pode tornar sua aprendizagem e implementação mais complexas para desenvolvedores iniciantes. Alguns recursos exigem um conhecimento mais aprofundado e prática para serem utilizados efetivamente.
  • Com à medida que um projeto web cresce e se torna mais complexo, a manutenção e organização do código CSS3 podem se tornar desafiadoras. Sem uma estrutura adequada, o código pode se tornar difícil de gerenciar e atualizar.

O que é CSS3?
CSS3 é uma linguagem de estilo utilizada para controlar a apresentação visual de um documento HTML.
CSS
CSS3 é a terceira versão do CSS (Cascading Style Sheets), uma linguagem de estilo utilizada para definir a apresentação e o design de documentos HTML. Lançado em 1999, o CSS3 introduziu uma série de recursos avançados que ampliaram as possibilidades de estilização e layout na web.

Grandes projetos que utilizam CSS3:
Airbnb, YouTube, Apple, Netflix, The New York Times, Pinterest.

javaScript.

Prós do JS:

  • JavaScript permite adicionar interatividade aos sites, respondendo a ações do usuário, realizando validações de formulários e atualizando conteúdos dinamicamente, proporcionando uma experiência mais rica e envolvente.
  • Suporta a criação de funções e objetos reutilizáveis, o que facilita o desenvolvimento de projetos complexos, permitindo a modularidade e a manutenção do código.
  • É suportado por todos os principais navegadores modernos e pode ser executado em diferentes sistemas operacionais. Isso torna a linguagem amplamente adotada e acessível.

Limitações do JS:

  • JavaScript pode ser vulnerável a ataques de segurança, como injeção de código malicioso e roubo de informações. É necessário tomar precauções adequadas para evitar essas vulnerabilidades.
  • Embora o JavaScript tenha melhorado significativamente em relação ao desempenho, sua execução ainda pode ser mais lenta do que outras linguagens de programação. Além disso, a execução de códigos complexos pode afetar negativamente o desempenho do site.
  • JavaScript depende do suporte do navegador e do ambiente do cliente. Isso significa que, se o usuário desabilitar o JavaScript em seu navegador, a funcionalidade do site pode ser afetada.

O que é JS?
JavaScript é uma linguagem de programação interpretada, que executa no lado do cliente (navegador), permitindo a criação de funcionalidades dinâmicas em páginas web.

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, amplamente utilizada no desenvolvimento de aplicativos web interativos. Ela permite adicionar funcionalidades dinâmicas aos sites, manipular elementos HTML, criar animações, validar formulários e interagir com serviços web. Abaixo, segue uma definição da tecnologia JavaScript, juntamente com três de seus principais benefícios, três de suas limitações e seis grandes projetos que a utilizam.

Grandes projetos que utilizam JS:
Google Maps, Facebook, Twitter, Netflix, Trello, Spotify.
